MITgcm.jl

Julia interface to MITgcm which allows user to not only read and analyze model output, but also download the source code, build the model executable, set run-time parameters, run simulations, verify model results, and perform other tasks related to MITgcm. These functionalities are documented via the Examples section.

Main Features

  • Read / Write / Scan the various MITgcm file types
  • Standard Modeling Worflow (ClimateModels' interface)
  • Standard MITgcm configurations (ECCO, verification)
  • Examples (docs, and notebooks in examples/)

Related packages

  • ClimateModels.jl : standard interface for modeling workflows.
  • OceanStateEstimation.jl : accessing and using ECCO solutions.
  • MeshArrays.jl : data structures for gridded model output.
  • IndividualDisplacements.jl : particle tracking applications and pathway simulationss.
